# Suprav Portfolio Starter v2 (Soothing + Accordion Projects)

## What changed vs v1
- Light, calm default theme (toggle to dark)
- Profile photo spot in the hero (replace `assets/profile.svg`)
- Projects are accordion / dropdown, so titles stay prominent
- About moved earlier (right after hero)
- Added Publications section with "Read paper" button
- Experience timeline cleaned up + added Incoming Gridomatix internship

## Run locally
Open `index.html` or use VS Code Live Server.

## Deploy (GitHub Pages user site)
Repo name must match your username:
`Suprav123.github.io` → URL: `https://suprav123.github.io`

## Replace placeholders
- `assets/RESUME_PLACEHOLDER.pdf` → your real resume (rename to keep link working)
- `assets/profile.svg` → your photo (jpg/png ok; update filename in `index.html` if you change it)
- LinkedIn + project links
